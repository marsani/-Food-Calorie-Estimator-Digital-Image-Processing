# -Food-Calorie-Estimator-Digital-Image-Processing
Application Created Vibe coding Using AntiGravity
interactive web application for estimating the calorie content of traditional foods (e.g., tahu, tempe, bakso) from a top-down 2D photo using a reference coin for scale calibration and image segmentation algorithms.
